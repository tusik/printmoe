# 这是什么		

这是一个功能强大的konachan web Api

他比konachan官方提供的api拥有更多的选项和功能

我们每天都会对konachan进行数据同步，截至2017年06月17日19：15分

我们收录了官方193649张图片的信息
	
## 如何实现		
		
我通过konachan提供的api将其所收录的所有图片信息保存进数据库，再通过Python编写接口并由Flask提供Web Api
